Interactive Elements

In-text link

You can find the finished game: here.

Sample Code

          <p>You can find the finished game: <a href = "#custom" target = "_blank">here.</a></p>

Button

CSS selector:

.button

Sample Code

<a href="#" class="button">Default Button</a>

Rendered element:

Default Button

Navigation Bar

CSS selector:

.headerNav

Home Games Art Music Citations

Sample Code

          <nav class = 'headerNav'>    
              <a href = '#home'>Home</a>     
              <a href = "#games">Games</a>
              <a href = "#arts">Art</a>
              <a href = "#music">Music</a>
              <a href = "#citations">Citations</a>
          </nav>

Submit / Reset Button

CSS selector:

.subBtn

Sample Code

              <form class = 'subBtn'>
                  <input type = 'submit' value = 'Submit'>
                  <input type = 'reset' value = 'Reset'>
              </form>

Text Input Field

CSS selector:

.textForm

Sample Code

              <form class = 'textForm'>
                  <label for = 'firstName'>First Name:</label>
                  <input type = 'text' id = 'firstName' name = 'firstName' placeholder = 'First Name'>
              </form>
