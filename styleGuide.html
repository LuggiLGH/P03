Interactive Elements

In-text link

You can find the finished game: here.

Sample Code

<p>You can find the finished game: <a href="#custom">here.</a></p>

Button

CSS selector:

.button

Sample Code

<button class="button">Default Button</button>

Rendered element:

Navigation Bar

CSS selector:

.headerNav (Sizes handled by .stickyHeader)

HomeGamesArtMusicCitations

Sample Code

<nav class='headerNav'> <a href='#home'>Home</a> <a href="#games">Games</a> <a href="#arts">Art</a> <a href="#music">Music</a> <a href="#citations">Citations</a> </nav>

Submit / Reset Button

CSS selector:

.subBtn

Sample Code

<form class='subBtn'> <input type='submit' value='Submit'> <input type='reset' value='Reset'> </form>

Text Input Field

CSS selector:

.textForm

Sample Code

<form class='textForm'> <label for='firstName'>First Name:</label> <input type='text' id='firstName' name='firstName' placeholder='First Name'> </form>

Modal Image

CSS selector:

.zoomImg, .modal, .modalImg

Sample Code

<img class='zoomImg' id='myImg' src='assets/img/characters/hones.webp' alt='A reference sheet for Hones, a skeleton caster/assassin.' onclick="openModal('modal1', 'A reference sheet for Hones, a skeleton caster/assassin.')"/> <div id='modal1' class='modal'> <span class="close" onclick="closeModal('modal1')">&times;</span> <img class="modalImg" id='hones' src='assets/img/characters/hones.webp' alt='A reference sheet for Hones, a skeleton caster/assassin.'> <div class="caption"></div> </div>
